# Button

<Props of={IconButton} />

###Цвет
color: oneOf(['primary', 'secondary']),
   
   
###Вариант использования
variant: PropTypes.oneOf(['text', 'textBlock', 'regular', 'regularBlock']),
  
  
###Размер
size: oneOf(['small', 'medium', 'large']),

```

```
