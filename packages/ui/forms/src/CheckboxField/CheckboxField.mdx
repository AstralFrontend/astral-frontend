# CheckboxField

<Props of={CheckboxField} />

## Props
  - disabled: bool
  - labelPlacement: oneOf(['end', 'start', 'top', 'bottom']),
  - className: string
  - label: node
  - name: string.isRequired

default:
  - labelPlacement = 'end'

## Basic code

```
<CheckboxField label="CheckboxField" name="CheckboxField" />
```
